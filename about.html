Hi there! My name is Ahmed and welcome to my learning journal.

I'm excited to start a new learning journey as a Scrimba Full-Stack Developer Career Path student. I've made the big jump from data science to web development to get expert code reviews of my projects and meet like-minded peers. Based in the UAE, I'm building modern web applications and documenting my progress along the way.

How I stay committed to learning

Building projects is my primary learning method. Rather than just watching tutorials, I dive into building real applications—from restaurant ordering systems to Instagram feed clones. Each project teaches me new concepts and reinforces what I've learned.

I maintain a consistent learning schedule, dedicating time each day to code, read documentation, and solve problems. The Scrimba platform's interactive approach has been transformative, allowing me to pause, experiment, and truly understand each concept before moving forward.

How I got started

My journey began in data science, working with Python, SQL, and Jupyter Notebooks on various analytics projects. While I enjoyed solving data problems, I realized I wanted to build things people could interact with directly—creating user experiences that solve real-world problems.

That's when I discovered Scrimba's Full-Stack Developer Career Path. The hands-on, project-based learning approach immediately resonated with me. Starting with foundational HTML and CSS, I've progressed to building interactive JavaScript applications, and I'm now working on more complex full-stack projects.

This learning journal documents my progress, the challenges I overcome, and the projects I build. Each post represents a milestone in my journey from data analyst to full-stack developer.
